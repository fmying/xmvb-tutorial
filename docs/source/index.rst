Tutorial of XMVB
===================================

**XMVB** is the abbreviation of `Xiamen Valence Bond <https://xacs.xmu.edu.cn/program/xmvb/>`. It focuses
on the *ab initio* valence bond computations based on nonorthorgonal atmic orbitals.


.. note::

   This project is under active development.
